# 🏫 LIL-CAMP: Lost Item Logger System

LIL-CAMP is a Python-based digital platform designed to digitize college campus lost-and-found tracking. It replaces traditional manual notice boards with a clean, modular terminal application featuring an automatic matching algorithm.

## ✨ Key Features
* **Automated Matching Algorithm:** Instantly cross-references newly found items with existing lost reports.
* **Persistent Storage:** Uses JSON file handling to ensure data is safely stored on the local drive and persists between sessions.
* **Input Validation:** Built-in date formatting (`DD-MM-YYYY`) and error handling (`try-except`) to prevent system crashes from user typos.
* **Object-Oriented Design:** Modular and clean architecture using standard OOP principles in Python.

## 💻 Tech Stack
* **Language:** Python 3.x
* **Libraries Used:** `json`, `os`, `datetime`
